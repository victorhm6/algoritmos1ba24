O que é Python
Python é uma linguagem de programação de alto nível, interpretada, de propósito geral e multiparadigma. Foi criada por Guido van Rossum e lançada pela primeira vez em 1991.
 Python se destaca por sua sintaxe limpa e legível, o que o torna fácil de aprender e usar. É conhecida por sua versatilidade e é amplamente utilizada em uma variedade de 
campos, como desenvolvimento web, ciência de dados, automação de tarefas, inteligência artificial, entre outros.

Terceiro commit:
Passo a passo fazer o download
1. Acesse o site oficial do Python: Abra um navegador da web e vá para o site oficial do Python em https://www.python.org/.
2. Baixe o instalador: No site, clique no link "Downloads" no menu principal.
3. Selecione a versão do Python: Dependendo da sua preferência, você pode escolher entre a versão mais recente do Python 3.x ou uma versão mais antiga do Python 2.x. Recomenda-se usar a versão mais recente do Python 3.x para novos projetos.
4. Escolha o instalador adequado para o seu sistema operacional: Se você estiver usando um sistema operacional Windows de 64 bits, provavelmente desejará baixar o instalador "Windows x86-64 executable installer". Se você estiver usando um sistema operacional Windows de 32 bits, baixe o instalador "Windows x86 executable installer".
5. Baixe o instalador: Clique no link de download apropriado para iniciar o download do instalador do Python.

Quarto commit:
Passo a passo de instalação
6. Execute o instalador: Após o download ser concluído, localize o arquivo do instalador em seu computador e execute-o clicando duas vezes sobre ele.
7. Inicie a instalação: No instalador, você terá a opção de adicionar o Python ao PATH do sistema. É recomendável selecionar essa opção para facilitar o uso do Python a partir da linha de comando. Em seguida, clique em "Install Now" para iniciar o processo de instalação.
8. Aguarde a conclusão da instalação: O instalador irá configurar o Python em seu sistema. Aguarde até que a instalação seja concluída.
9. Verifique a instalação: Após a conclusão da instalação, você pode verificar se o Python foi instalado corretamente abrindo o prompt de comando e digitando python --version. Isso deve exibir a versão do Python que foi instalada.

Como configurar virtuallanv pelo python:
1.Primeiro, certifique-se de ter o Virtualenv instalado em seu sistema. Caso não tenha, você pode instalá-lo usando o comando pip install virtualenv.
2.Após a instalação, crie um novo diretório para o seu projeto e navegue até ele usando o terminal ou prompt de comando.
3. Dentro do diretório do projeto, crie um novo ambiente virtual usando o comando virtualenv nome_do_ambiente. Substitua “nome_do_ambiente” pelo nome que deseja dar ao seu ambiente virtual.
4. Após a criação do ambiente virtual, ative-o utilizando o comando apropriado para o sistema operacional em que você está trabalhando.
